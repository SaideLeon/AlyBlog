# **App Name**: Kizomba Blog

## Core Features:

- Content Management: Create, edit, and delete blog posts, managing categories and tags for organization.
- Audio Player: Integrated audio player to stream Kizomba music and albums.
- Interactive Quizzes: Engage users with quizzes on Kizomba knowledge and artist trivia.
- Fan Feedback: Gather testimonials and reviews, moderating a message board for fan interaction.
- AI-Powered Content Suggestions: Utilize a tool that leverages user-generated comments and available material, offering post topic suggestions to administrators to potentially enhance blog engagement and reach.
- Contact Form: Allow users to send inquiries, booking requests, and other messages directly.

## Style Guidelines:

- Primary color: Vibrant red (#E63946) to reflect the energy of Kizomba.
- Background color: Light pink (#F1FAEE), a very desaturated tone of the primary color to maintain a vibrant style while preventing eye strain.
- Accent color: Orange-red (#F45B69), for call-to-action buttons and highlighting key elements.
- Headline font: 'Playfair', a serif with an elegant and high-end feel. Body Font: 'PT Sans', a humanist sans-serif for body.
- Use clean and modern icons related to music, culture, and user interaction.
- Responsive grid layout for a consistent user experience across devices.
- Subtle transitions and animations for interactive elements to enhance user engagement.
